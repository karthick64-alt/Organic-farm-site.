## Getting Started

1. Clone or download the project
2. Open `index.html` in a web browser
3. All files are self-contained (no build process required)

## Customization

### Colors
Edit CSS variables in `css/style.css`:
```css
:root {
    --color-green-primary: #5a8f3a;
    --color-brown-primary: #6d4c41;
    /* ... */
}
```

### Content
- Update product information in `products.html`
- Modify farm story in `about.html`
- Edit blog posts in `blog.html`
- Update contact information in footer sections

### Images
Replace placeholder images (currently using Unsplash) with your own farm photography.
